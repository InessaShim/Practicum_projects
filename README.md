# Проекты Яндекс Практикума

В этом репозитории собраны учебные проекты, выполненные в рамках программы обучения Яндекс Практикума по анализу данных. Каждый проект содержит код, данные и подробное описание.

## Навыки, продемонстрированные в проектах

* **Анализ данных:** Pandas, NumPy, разведочный анализ (EDA).
* **Визуализация:** Matplotlib, Seaborn.
* **Базы данных:** SQL (PostgreSQL), работа с DBeaver.
* **Инструменты:** Jupyter Notebook, Python.

## Как работать с репозиторием

1. Перейдите в папку нужного проекта (кликните на ссылку в списке проектов ниже).
2. Ознакомьтесь с `README.md` внутри папки проекта — там есть инструкция по запуску и описание результатов.
